Informācija par izstrādāto sistēmu

Šajā sadaļā jāapraksta, kas ir šī programma. Tavā gadījumā tā ir konsoles Java lietotne automašīnu kolekcijas pārvaldīšanai, kur lietotāji var pierakstīties sistēmā un saglabāt savas iecienītākās automašīnas savā profilā. Šajā sadaļā var iekļaut:
Īss funkcionalitātes apraksts
Konsoles lietotnes ekrānuzņēmumi
Iespējamās priekšrocības salīdzinājumā ar analogiem

Sistēmas lejupielādes adrese

Šeit būs saite programmas lejupielādei. Ja programma vēl nav pieejama tiešsaistē (piemēram, GitHub vai Google Drive), var uzrakstīt "Drīzumā pieejams". Šajā sadaļā var iekļaut:
Pogu "Lejupielādēt"
Aprakstu par to, kas tieši tiks lejupielādēts (arhīvs ar .jar failu, avota kods utt.)
Sistēmas prasības (piemēram, nepieciešama Java 17)

Lietotāja ceļvedis

Šajā sadaļā jābūt paskaidrojumam par to, kā lietot programmu. Tā kā tā ir konsoles lietotne, var iekļaut:
Komandu sarakstu un to aprakstu (piemēram, add car, delete car, list cars)
Konsoles darbības piemēru
Instalēšanas ieteikumus, ja tas ir nepieciešams
